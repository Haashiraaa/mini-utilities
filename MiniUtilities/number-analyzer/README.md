#  Number Analyzer

A simple Python CLI tool that helps you analyze numbers in different ways:
-  Check if a number is **prime**
-  Check if a number is **even or odd**
-  Find the **square root** of a number

This is a beginner-friendly project and can be extended into something bigger.

---

##  Installation

1. Clone the repository:

```bash
git clone https://github.com/Haashiraaa/number-analyzer.git
cd number-analyzer
   
2. Make sure the following files are in the same directory so imports work correctly:

number_analyzer.py
prime_checker.py
even_odd_checker.py
square_root_checker.py


3. Run the program with:

python number_analyzer.py
