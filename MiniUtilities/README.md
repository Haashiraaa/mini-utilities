# MiniUtilities

A collection of small utility scripts and tools designed to simplify everyday tasks. Each script in this repository aims to solve a specific problem, making it easier to enhance productivity.

## Features

- Modular utilities for common programming tasks
- Easy to integrate and extend
- Lightweight and minimal dependencies

## Usage

1. Clone the repository:
   ```
   git clone https://github.com/Haashiraaa/mini-utilities.git
   ```
2. Navigate to the `MiniUtilities` folder:
   ```
   cd mini-utilities/MiniUtilities
   ```
3. Run or import the desired utility script based on your needs.
